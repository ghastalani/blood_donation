Blood Donation Platform

A simple web platform that helps connect blood donors with people in need.
The goal of this project is to make it easier to request blood and find available donors quickly.

Features

- User registration and login
- Request blood donations
- Search for available donors
- Simple and responsive interface

Technologies Used

- React (Frontend)
- Supabase (Database & authentication)
- PHP (Backend API)

Purpose

This platform aims to support communities by making blood donation more organized and accessible.

Author

Developed as a learning project in web development.
